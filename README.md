## Casos de trabajo

### 1) Costurero Digital — E-commerce
**Problema:** experiencia de compra mejorable en móvil.  
**Solución:** UI más clara, mejor estructura de navegación y optimización de carga.  
**Resultado:** mejora de conversión y experiencia de usuario.

### 2) FotoNico Studio — Portfolio visual
**Problema:** sitio pesado y navegación poco fluida.  
**Solución:** carga progresiva, animaciones con propósito y arquitectura más limpia.  
**Resultado:** mejor retención y experiencia más profesional.
